Пояснення атрибута align у HTML

Щоб розмістити текст зліва,по центру або справа на сторінці ми використаємо атрибут align.

<h1 align="left"></h1>
<h1 align="center"></h1>
<h1 align="right"></h1>

Атрибут align записується у відкриваючому тезі і може використовуватись для тегів p та h

Спробуємо змінити колір нашого абзацу
Колір тексту - властивість(атрибут) color
Також нам хочеться виділити текст певним кольором.
Зробити це можна за допомогою властивості color, та наявності тексту на нашій сторінці.

Атрибут class в тегах - селектор для style
Задає стильовий клас, який дозволяє зв'язати певний тег зі стильовим оформленням. У значенні допускається вказувати відразу кілька класів, розділяючи їх між собою пробілом.

Клас(class) - властивість(атрибут, селектор) для стилізацї.

Стиль шрифту(Сім'я шрифтів) - font-family та розмір тексту - font-size
font-family - встановлює сімейство шрифту, яке буде використовуватися для оформлення тексту вмісту.
font-size - визначає розмір шрифту(тексту) елемента.
